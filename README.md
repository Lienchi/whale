# WHALE DONE

人性化 CRM 平台

## 說明

以下的說明將為您提供該項目的一個副本，讓您的本地電腦上運行以利於開發和測試目的。


## 必備條件

一顆強大的心（誤 XDD）

## 基本條件

您的電腦至少需要有這些軟體

* 檢視網頁用的 Chrome 瀏覽器
* Terminal
* 文字編輯器。例如：Sublime Text 


## 安裝

您也許需要安裝這些軟體

* Homebrew
* Ruby
* Rails


## 螢幕截圖



## 如何開始

1. Run `bundle install`
2. Run `rails db:migrate`
3. Run `rails db:seed`


## 建置

- Rails '~> 5.1.5'
- Bootstrap3
